'How Happy Are We?' Analysing Sentiment Within Twitter Data

Completed: June 2016-May 2017 (Final-Year Project & Dissertation)

A mobile application which analyses the happiness of friends and other followed accounts on a given day.
Users are placed into a group representing their happiness (ranging from Very Negative to Very Positive). These can be selected, along with users and tweets, to provide a summary of the analysis completed.
Attempts an implementation of an equation proposed by Adam Kramer, which normalises a user's happiness relative to all other users on the day. This is the idea that a user is 'only remarkably positive if they are positive on the day in question'.
The result is an application providing more personal, relevant results than already existing applications.

Tools: Java, Twitter4J, Android Studio
